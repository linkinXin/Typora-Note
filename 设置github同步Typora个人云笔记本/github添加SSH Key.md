# github添加SSH Key

###### ### 个人学习整理，仅供参考

> 登录github账号，进入Settings

![image-20210222225033141](../assets/image-20210222225033141.png)

> 选择 "SSH and GPG keys" >> "New SSH key"

 ![image-20210222225443633](../assets/image-20210222225443633.png)

**Tip:  粘贴的Key是用指令复制的，直接复制文件是无效的**
